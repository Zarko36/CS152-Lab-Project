# CS152_Programming_Language_All_Phases
Proramming Language Name: Bb


### Language Features

| Language Feature      | Code Example |
|-----------------------|--------------|
| Variable Declaration  | int x;       |
| Add                   | x + y        |
| Sub                   | x - y        |
| Multiply              | x * y        |
| Divide                | x / y        |
| Mod                   | x % y        |
| Less Than             | x < y        |
| Less Than Equal       | x <= y       |
| Equality              | x == y       |
| Not Equality          | x != y       |
| Greater Than          | x > y        |
| Greater Than Equal    | x >= y       |
| Write                 | print(x)     |
| Read                  | read(x)      |
| Arrays                | int [8] array|
| Comments              | // This is a comment|
